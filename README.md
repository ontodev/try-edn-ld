# Try EDN-LD

Try EDN-LD is an online Clojure REPL for experimenting with [EDN-LD](https://github.com/ontodev/edn-ld). It is adapted (with gratitude!) from [TryClojure!](http://tryclj.com).

## Usage

http://try.edn-ld.com

To run it locally, use `lein ring server`.
